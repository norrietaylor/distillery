Distillery Settings

Configure your connection to the Distillery MCP server

Server Configuration

URL of the remote Distillery MCP server. Default: https://distillery-mcp.fly.dev/mcp

Port for local Distillery MCP server. Default: 8000

Automatically detect and connect to localhost when available. Falls back to remote server if not found.

Content Configuration

Pre-fill the project field in bookmarks. Leave blank to prompt on each save.

Comma-separated tags to apply to bookmarks. Example: research, browser-saved

Authentication

For remote server authentication. Optional if using local server or public demo. Required for GitHub OAuth flow on custom remote instances.
