## Summary
Three standalone async helper functions extracted from quality.py:
- `run_dedup_check(store, classification_config, content)` -- returns dedup result dict
- `run_conflict_discovery(store, threshold, content)` -- returns conflict candidates dict
- `run_conflict_evaluation(store, threshold, content, llm_responses)` -- returns conflict result dict

All helpers accept typed arguments and return typed dicts, independent of MCP handler
signatures. The existing `_handle_check_dedup` and `_handle_check_conflicts` handlers
now delegate to these helpers. All 65 tests pass (21 new + 44 regression).
