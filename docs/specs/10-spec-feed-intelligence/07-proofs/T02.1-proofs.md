## Implementation

Added three functions to `src/distillery/feeds/poller.py`:

1. `build_keyword_map(vocabulary: dict[str, int]) -> dict[str, str]`
   - Extracts leaf segments from tag paths and maps them to full tag paths
   - Splits hyphenated leaves into words longer than 3 chars
   - Resolves collisions by occurrence count (higher wins), then alphabetically

2. `match_topic_tags(text: str, keyword_map: dict[str, str]) -> list[str]`
   - Case-insensitive word tokenization of input text
   - Looks up each token in the keyword map
   - Returns deduplicated list of matched full tag paths

3. `derive_all_tags(item: FeedItem, source_type: str, keyword_map: dict[str, str]) -> list[str]`
   - Combines Tier-1 source tags and Tier-2 topic tags
   - Deduplicates across tiers; source tags appear first
