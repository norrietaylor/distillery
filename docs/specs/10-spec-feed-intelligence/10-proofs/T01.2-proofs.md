**Status**: COMPLETED

## Implementation Summary

Added `_derive_source_tags(item: FeedItem, source_type: str) -> list[str]` to
`src/distillery/feeds/poller.py` and wired it into `_item_to_entry_kwargs()`.

Tag derivation rules implemented:
- All feeds: `source/{source_type}` (e.g. `source/rss`, `source/github`)
- GitHub feeds: `source/github/{owner}/{repo}` via `_parse_github_url()`
- Reddit RSS: `source/reddit/{subreddit}` parsed from URL path
- Other RSS: `source/{domain-slug}` with `www.` stripped and dots replaced by hyphens
  (required for `validate_tag()` compatibility, which only allows `[a-z0-9-]` per segment)

All derived tags are validated via `validate_tag()`; invalid tags are dropped silently
with a `logger.debug()` call.
