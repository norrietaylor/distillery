## Implementation Summary

Added `distillery retag` subcommand to `src/distillery/cli.py`:

- Parser: `retag` subcommand with `--dry-run` and `--force` flags
- Handler: `_cmd_retag()` function
  - Loads tag vocabulary via `store.get_tag_vocabulary()`
  - Builds keyword map via `build_keyword_map()` once per cycle
  - Iterates feed entries in batches of 100 (`_RETAG_BATCH_SIZE = 100`)
  - Skips already-tagged entries unless `--force`
  - Applies `derive_all_tags()` to reconstruct FeedItem from entry metadata
  - Updates via `store.update(entry_id, {"tags": new_tags})` (no re-embedding)
  - Reports `total_scanned` and `total_updated`
- Dispatch: Added `elif command == "retag"` branch in `main()`
