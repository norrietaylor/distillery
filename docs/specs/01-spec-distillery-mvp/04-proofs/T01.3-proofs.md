### Design Decisions

- Used Python `dataclasses` with sensible defaults for all config sections
- `load_config()` accepts an optional explicit path; falls back to
  `DISTILLERY_CONFIG` env var, then `distillery.yaml` in cwd
- Missing config file returns all defaults (no error) for easy local dev
- Explicit path that does not exist raises `FileNotFoundError`
- Invalid `embedding.provider` values raise `ValueError` with clear message
- `confidence_threshold` is validated to [0.0, 1.0] range
- `embedding.dimensions` is validated to positive integer
