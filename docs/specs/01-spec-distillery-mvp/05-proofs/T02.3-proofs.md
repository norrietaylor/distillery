## Implementation

Implemented the following methods on `DuckDBStore` in `src/distillery/store/duckdb.py`:

- `store(entry)` - Inserts an Entry into DuckDB with a zero-vector embedding placeholder.  Returns the entry's UUID string.
- `get(entry_id)` - Fetches an entry by ID; returns `None` if not found.
- `update(entry_id, updates)` - Applies a partial update; rejects `id`, `created_at`, `source`; increments `version`; refreshes `updated_at`.  Raises `KeyError` if entry not found.
- `delete(entry_id)` - Soft-deletes by setting `status = 'archived'`.  Returns `True` if found, `False` otherwise.

All methods are `async` and delegate synchronous DuckDB work to `asyncio.to_thread()`.
