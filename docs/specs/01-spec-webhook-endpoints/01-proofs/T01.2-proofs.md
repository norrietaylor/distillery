## Implementation Summary

Created `src/distillery/mcp/webhooks.py` with:

- `create_webhook_app()` factory returning a Starlette app with 3 POST routes (`/poll`, `/rescore`, `/maintenance`)
- Bearer token auth via `hmac.compare_digest` against configured env var
- 401 response with `{"ok": false, "error": "unauthorized"}` on auth failure
- Per-endpoint cooldown tracking via `store.get_metadata`/`store.set_metadata` with ISO 8601 timestamps
- 429 response with `Retry-After` header and `{"ok": false, "error": "too_early", "retry_after": N}` within cooldown window
- Default cooldowns: poll=300s, rescore=3600s, maintenance=21600s
- `_ensure_store()` async helper replicating MCP lifespan init logic
- `RateLimitMiddleware` applied with 10 req/min, 100 req/hr limits
- Stub route handlers returning `{"ok": true, "data": {}}`
- All code passes `mypy --strict` and `ruff check`
