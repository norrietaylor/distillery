## Implementation Summary

### server.py change
Added `server._distillery_shared = _shared  # type: ignore[attr-defined]` immediately after
`server = FastMCP(...)` at line 386. This exposes the shared state dict so `__main__.py`
can pass it to `create_webhook_app`.

### __main__.py change
Added conditional webhook app composition in the HTTP branch after `wrapped_app` is built:
- Imports `ASGIApp` from `starlette.types` for correct typing
- If `config.server.webhooks.enabled` is True AND the secret env var is set:
  - Imports `create_webhook_app` from `distillery.mcp.webhooks`
  - Creates `webhook_app = create_webhook_app(server._distillery_shared, config)`
  - Composes `Starlette(routes=[Mount("/api", app=webhook_app), Mount("/", app=wrapped_app)])`
  - Assigns composed app as `final_app`
- Otherwise: `final_app = wrapped_app` (unchanged behavior)
- Mount("/api") appears before Mount("/") ensuring correct prefix matching
