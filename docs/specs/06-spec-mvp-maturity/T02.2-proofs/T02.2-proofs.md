## Tool Behaviour

`distillery_stale` queries entries where `COALESCE(accessed_at, updated_at) < NOW() - INTERVAL N DAYS`:
- Only non-archived entries are included
- Results ordered stalest-first (ASC by last_accessed)
- Parameters: `days` (default: config.classification.stale_days), `limit` (default: 20), `entry_type` (optional filter)
- Returns: `days_threshold`, `entry_type_filter`, `stale_count`, `entries[]` (with id, content_preview, entry_type, author, project, last_accessed, days_since_access)
