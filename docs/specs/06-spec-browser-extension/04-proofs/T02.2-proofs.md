**Status**: COMPLETED

## Implementation

Created `browser-extension/src/content.js` which:

- Runs on all pages at `document_end` (after `vendor/readability.js`)
- Extracts `document.title`, `window.location.href`
- Extracts `<meta name="description">` content
- Extracts Open Graph tags: `og:title`, `og:description`, `og:image`
- Extracts full article text via `new Readability(document.cloneNode(true)).parse()`
- Captures selected text via `window.getSelection().toString()`
- Listens for `chrome.runtime.onMessage` with `type: "extractContent"`
- Returns structured data object to caller
