## Implementation

### Files Modified

- `browser-extension/src/popup.html` — Replaced placeholder with full bookmark form
- `browser-extension/src/popup.js` — Added bookmark tab initialization and save flow
- `browser-extension/src/popup.css` — Added bookmark form styles with dark mode support
- `browser-extension/src/background.js` — Added `bookmark` message handler

### Key Decisions

1. **Bookmark tab is active by default** — tab bar reordered: Bookmark, Watch, Status
2. **Content extraction fallback** — if content script unavailable (chrome:// pages etc.), falls back to tab title/url from chrome.tabs API
3. **Author resolution** — GitHub username from connection state first, then `defaultAuthor` from storage
4. **Offline support** — network errors enqueue via `offlineQueue`, response has `status: 'queued'`
5. **selectedText priority** — if user has text selected, it is used as body over Readability articleText
