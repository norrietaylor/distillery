Handler Flow:
1. Verify menuItemId is 'save-to-distillery'
2. Extract content from active tab using chrome.tabs.sendMessage('extractContent')
3. Fallback to basic tab metadata if content script unavailable
4. Load default tags, project, author from chrome.storage.local
5. Determine final author (prefer logged-in username over local author)
6. Build markdown-formatted bookmark content using buildBookmarkContent()
7. Call mcpClient.callTool('distillery_store', args)
8. On success: showContextMenuSuccess() - badge "✓" green for 2s
9. On MCPNetworkError: queue operation, show success (queued)
10. On other errors: showContextMenuError() - badge "!" red

- Content extraction with fallback for restricted pages
- Respects existing settings (tags, project, author)
- Integrates with offline queue for network failures
- Shows user feedback via badge and desktop notifications
