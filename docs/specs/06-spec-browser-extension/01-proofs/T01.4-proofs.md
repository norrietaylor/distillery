## Summary

`background.js` implements the Manifest V3 service worker with:

- **Auto-detect**: `probeLocal()` uses AbortController with 1-second timeout to probe `localhost:{port}/mcp`; falls back to remote URL on failure.
- **Periodic re-probe**: `chrome.alarms` API (5-minute period) survives service worker suspension.
- **Options reactivity**: `chrome.storage.onChanged` listener triggers `connect()` when relevant options change.
- **Shared MCPClient**: Single instance created via `importScripts('mcp-client.js')`, disconnected and re-initialized on each reconnect.
- **Message API**: Handles `getConnectionStatus`, `reconnect`, `callTool`, `OPTIONS_UPDATED`, `setAuthToken` from popup and content scripts.
- **Connection state**: Broadcasts `{ connected, serverType, serverUrl, username, queuedOperations }` to all extension views after each state change.
- **First-install**: `chrome.runtime.onInstalled` opens options page when `reason === 'install'`.
- **Auth error handling**: 401 responses clear stored token and broadcast disconnected state.
