- Green indicator dot with glow effect
- Label: "Connected"
- Server type: "(local)" or "(remote)"

**Disconnected State**:
- Red indicator dot with subtle glow
- Label: "Disconnected"
- Message: "Cannot reach server. Check settings or try again."

### Tab System

Three tabs implemented with placeholder content:
1. **Status Tab** (active by default)
   - Connection status message
   - Offline queue indicator (if pending operations exist)

2. **Bookmark Tab** (placeholder for T02)
   - Will include page title, description, tags, project selector, save button

3. **Watch Tab** (placeholder for T04)
   - Will include detected feeds, watch buttons, watched sources list

### Integration Points

**Communication with Background Worker**:
- Sends `getConnectionStatus` message to background worker
- Receives connection state: `{ connected, serverType, serverUrl, username, queuedOperations }`
- Listens for `connectionStatusChanged` broadcasts from background worker

**Auto-Refresh**:
- Queries connection status every 5 seconds
- Updates UI immediately if status changes
- Can also receive push notifications from background worker

**Settings Integration**:
- Settings button opens options page (T01.3)
- Will integrate with stored settings in chrome.storage.local
