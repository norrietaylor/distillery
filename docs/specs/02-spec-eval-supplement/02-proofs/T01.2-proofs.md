## Implementation Details

### File Created
- `.github/workflows/eval-pr.yml` (38 lines)

### Workflow Structure
```yaml
name: Eval (PR)
on:
  pull_request:
    branches:
      - main

jobs:
  eval:
    runs-on: ubuntu-latest
    timeout-minutes: 5
    steps:
      1. Checkout repository
      2. Set up Python 3.12 (with pip cache)
      3. Set up Node.js 20
      4. Install Distillery dev dependencies
      5. Run promptfoo eval
```

### Key Features
- **Trigger**: Pull request events targeting main branch
- **Runner**: ubuntu-latest (consistent with eval-nightly.yml)
- **Timeout**: 5 minutes (sufficient for <2 minute smoke tests)
- **Permissions**: contents:read, id-token:write (for OAuth token)
- **Environment Variables**: 
  - CLAUDE_CODE_OAUTH_TOKEN (GitHub secret)
  - CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC=1
- **Command**: `npx promptfoo@latest eval --config promptfooconfig.yaml`

### Pattern Compliance
Follows established patterns from `.github/workflows/eval-nightly.yml`:
- Same Python 3.12 and Node.js 20 versions
- Same setup actions and caching strategy
- Same CLAUDE_CODE_OAUTH_TOKEN secret
- Same CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC flag
- Same repository checkout and installation pattern
- ubuntu-latest runner consistent with nightly workflow
